## 🎯 ZenBlock 是什么？

这是一个**不讲道理的强制拦截系统**！

- 🚫 **强制拦截**: 想打开B站？先来反思页面冷静一下
- 📊 **数据打脸**: 用可视化图表告诉你今天摸了多少次鱼
- 🧘 **禅意语录**: 被拦截时送你一句斯多葛哲学名言
- 🌓 **深色模式**: 半夜偷偷摸鱼也能保护眼睛（并不鼓励）
- 🌍 **双语支持**: 中英文随意切换，国际友人也能用

## 🚀 一键启动（懒人专用）

### Windows 用户

#### 开发模式（调试用）
```cmd
双击 start.bat
```
就这么简单！脚本会自动：
1. 检查并安装依赖
2. 同步数据库
3. 启动开发服务器
4. 打开浏览器访问 http://localhost:3000/zh

#### 生产模式（正式使用）
```cmd
双击 deploy.bat
```
会自动完成：
1. 安装生产依赖
2. 设置数据库
3. 构建优化版本
4. 启动生产服务器

### Linux/Mac 用户

#### 开发模式
```bash
chmod +x start.sh
./start.sh
```

#### 生产模式
```bash
chmod +x deploy.sh
./deploy.sh
```

## 🎮 系统托盘版（终极懒人方案）

### 首次安装

```cmd
# Windows
zenblock-cli.bat install

# Linux/Mac
./zenblock-cli.sh install
```

安装完成后，ZenBlock 会：
- 🎯 开机自动启动
- 📌 驻留在系统托盘
- 🔄 后台静默运行

### 托盘功能

右键点击托盘图标：
- **打开控制台** - 跳转到主页面
- **查看仪表盘** - 查看数据统计
- **启动/停止服务** - 控制服务运行
- **开机自启** - 切换自启动状态
- **退出** - 关闭服务

### 卸载服务

```cmd
# Windows
zenblock-cli.bat uninstall

# Linux/Mac
./zenblock-cli.sh uninstall
```

## 📖 使用指南

### 第一步：生成拦截脚本

1. 访问 http://localhost:3000/zh
2. 输入你想戒掉的网站（比如 `bilibili.com`）
3. 点击"生成拦截脚本"
4. 点击"一键安装"下载脚本

### 第二步：安装脚本

浏览器会自动识别 `.user.js` 文件：
- 如果安装了 Tampermonkey，会自动弹出安装界面
- 如果安装了 ScriptCat，也会自动识别

**没有油猴插件？**  
先安装 [Tampermonkey](https://www.tampermonkey.net/) 或 [ScriptCat](https://docs.scriptcat.org/)

### 第三步：体验"被拦截的快乐"

现在试着打开 B站：
1. 页面会被强制拦截
2. 跳转到反思页面
3. 看到今天的摸鱼次数
4. 收到一句禅意语录
5. 自我反省（大概）

### 第四步：查看数据

访问仪表盘页面：
- 📈 今日摸鱼次数
- 🕐 薄弱时段分布（哪个时间段最容易摸鱼）
- 🔥 30天热力图（一个月的摸鱼历史）
- 📊 按网站分类统计

## 🎨 功能特性

### 😈 核心功能

- **零容忍拦截**: 想打开就拦截，没有"确认是否继续"
- **强制反思**: 每次拦截都记录，让你看清自己
- **数据可视化**: 用图表暴露你的摸鱼真相
- **多网站支持**: B站、抖音、小红书...一网打尽

### 🎭 特色功能

- **主题切换**: 亮色/暗色随心切
- **语言切换**: 中英文下拉选择
- **禅意语录**: 拦截时的心灵鸡汤（没啥用但很有格调）
- **系统托盘**: 后台运行，不打扰

### 🔒 隐私保护

- **本地数据**: 所有数据存在本地 SQLite
- **完全离线**: 不联网，不上传
- **零追踪**: 不收集任何个人信息

## 🛠️ CLI 命令大全

```bash
# 安装服务（开机自启）
zenblock-cli install

# 卸载服务
zenblock-cli uninstall

# 启动服务

# 打开仪表盘
zenblock-cli dashboard
```

## 🤔 常见问题

### Q: 脚本不生效怎么办？
A: 检查是否安装了油猴插件，脚本是否启用。

### Q: 能绕过拦截吗？
A: 能，但那样就失去意义了。自律的前提是诚实面对自己。

### Q: 数据会上传吗？
A: 不会！所有数据都在本地，完全离线运行。

### Q: 为什么叫 ZenBlock？
A: Zen（禅）+ Block（拦截） = 在拦截中寻找内心的平静 🧘

### Q: 能拦截手机吗？
A: 目前只支持电脑浏览器。手机？建议直接卸载App（最有效）。

### Q: 会影响正常工作吗？
A: 不会。只拦截你设置的网站，其他网站正常访问。
